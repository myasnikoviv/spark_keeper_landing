❗ ДОДАТКОВИЙ КРИТИЧНИЙ БАГ — CTA IDLE ANIMATION

Проблема

У всіх CTA (особливо Start Capturing, фінальний Get Spark Keeper / WhisperedCTA):
	•	Idle-анімація відсутня або майже непомітна
	•	Ефект зʼявляється тільки на hover
	•	Через це CTA:
	•	виглядає “мертвим”
	•	губиться в середовищі
	•	не читається як головна дія

Це протирічить базовій ідеї лендингу, де:

нічого не має бути повністю статичним

⸻

✅ ВИМОГА (ОБОВʼЯЗКОВО ВИКОНАТИ)

1️⃣ Idle state (без наведення)

Кожен CTA ЗАВЖДИ має жити, навіть коли курсор не наведений:
	•	Pulse / breathing animation (scale / glow / opacity)
	•	Період: ~3–4 секунди
	•	Амплітуда: достатньо помітна, але не агресивна
(не “ледь видно”, а так, щоб око ловило рух)

❗ Якщо idle-пульсацію не видно з першого погляду — це вважається багом.

⸻

2️⃣ Hover state (посилення, а не єдиний ефект)

Hover НЕ замінює idle-анімацію, а підсилює її:
	•	Idle → Hover = більше glow / яскравіший колір / швидший pulse
	•	Hover не є єдиним моментом життя кнопки

⸻

3️⃣ WhisperedCTA (фінальний)

Для WhisperedCTA:
	•	Idle:
	•	мʼякий, “дихаючий” glow
	•	легкий shimmer / spark pass-through (раз на кілька секунд)
	•	Hover:
	•	короткий, але відчутний “energy spike”

Це головний CTA сторінки, він не може бути тихішим, ніж інші.

⸻

4️⃣ Acceptance Criteria (чітко)

CTA вважається виправленим, якщо:
	•	✅ Idle-анімація помітна без hover
	•	✅ Hover підсилює, а не “вмикає” ефект
	•	✅ На темному фоні CTA не губиться
	•	❌ Якщо “пульсація є, але її майже не видно” — НЕ приймається
